## 🎯 HOW TO USE NEW FEATURES

### For Users:

1. **Book with Photos:**
   - Go through booking flow
   - On step 4 (Details), upload photos of the task
   - Add recurring schedule if needed

2. **Pay with Paystack:**
   - After booking confirmation
   - Choose payment method (Card/Bank/Wallet)
   - Complete payment securely
   - Get payment reference

3. **Send Images in Messages:**
   - Open conversation with tasker
   - Click image icon
   - Select images to share
   - Send with or without text

4. **Tip Your Tasker:**
   - After task completion
   - Click "Add Tip" button
   - Choose amount or enter custom
   - Confirm payment

### For Taskers:

1. **View Earnings:**
   - Go to Tasker Dashboard
   - Click "Earnings" tab
   - See charts and breakdown
   - Filter by week/month/year

2. **Check Calendar:**
   - Go to Tasker Dashboard
   - Click "Calendar" tab
   - Navigate months
   - See all scheduled tasks

3. **Accept Jobs:**
   - Jobs tab shows available work
   - Accept or decline
   - Track assigned jobs
   - Update status

## 🔑 IMPORTANT NOTES

### Paystack Integration:
- Replace `pk_test_your_public_key_here` in `PaystackPayment.jsx` with your actual Paystack public key
- Test mode uses test keys
- Live mode requires live keys from Paystack dashboard

### Image Storage:
- Currently stores images as base64 in localStorage
- For production, use cloud storage (AWS S3, Cloudinary)
- Implement image compression for better performance

### Recurring Bookings:
- Currently stores recurring data but doesn't auto-create future bookings
- Implement cron job or scheduled task to create recurring bookings

### Tips:
- Tips are added to booking total
- Payment for tips should be processed separately
- Consider platform fee on tips
